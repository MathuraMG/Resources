In case you are using github for documentation (and NOT for code versioning - you can follow the below steps)

Below are the steps for creating a repo on Github and subsequently adding files and folders to it

1. Create a github account - https://github.com/
2. Once you created an account, we have to create a new repo. Click on the plus sign in the top right corner, and select " new repository"
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-1.png" alt="adding new repo" width="400"/>
3. Go ahead and name your repository, and then click the button "Create Repository"
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-2.png" alt="" width="400"/>
4. Click on the link "Creating a new file"
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-3.png" alt="" width="400"/>
5. Name your file and type in it
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-4.png" alt="" width="400"/>
6. Once you are done putting in the content, click on "Commit new file"
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-5.png" alt="" width="400"/>
7. You should see that file when you refresh your repository, you can click on the pencil to edit the file
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-6.png" alt="" width="400"/>
8. You can also upload files and folders and images using the "Upload files" option
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-7.png" alt="" width="400"/>
<img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-8.png" alt="" width="400"/>
9. And you're all set!

* To delete a file, open the file, and click on the trash can symbol on the top right
* <img src="https://github.com/MathuraMG/Resources/blob/main/media/githubForDocumentation/gh-9.png" alt="" width="400"/>
